## 為什麼開發這個工具？(The "Why")

市面上許多 QR Code 服務商（如 QRFY 等）使用的是「動態 QR Code」。他們生成的圖片並不直接包含你的網址，而是指向他們的中繼伺服器。這意味著他們掌握了控制權——一旦你不續費訂閱，你的 QR Code 就會失效，這對已經印刷的商業物料來說是巨大的風險。

**本工具旨在解決這個問題：**
我們生成的是**「靜態 QR Code」**。網址資料直接編碼在圖片的黑白點陣中。一旦生成，這張圖片就物理性地永久包含了該網址，不需要依賴任何第三方伺服器，永遠不會過期，真正實現「一次生成，終身有效」。

### 🖨️ 智慧型印刷輸出策略
程式會根據你選擇的格式自動調整輸出邏輯，確保印刷品質：

| 格式 | 適用場景 | 智慧邏輯 |
| :--- | :--- | :--- |

## 使用指南 (How to Use)

1.  **輸入網址**：貼上您希望 QR Code 指向的完整目標網址（例如 `https://memory8.com`）。
2.  **自訂外觀**：調整您喜歡的顏色。
3.  **背景設定**：若需要去背圖給設計師使用，請勾選「透明背景」。
4.  **選擇格式並下載**：
    * 一般使用推薦選 **PNG**。
    * 若廠商要求 AI/EPS 檔，請下載 **SVG** 給他們。

## 驗證指南：如何確認 QR Code 是有效的？

為了確保您生成的 QR Code 是可讀的，且確實是「靜態直連」，我們建議進行以下驗證。

### 重要提示：關於「透明背景」的誤區

如果您下載了「透明背景」的 PNG，在某些看圖軟體（尤其是深色模式下）或線上掃描器中打開時，可能會看到**一片全黑**。

* **原因**：這是正常的視覺現象。黑色的 QR Code 點點疊加在軟體預設的黑色背景上，導致視覺上消失。
* **解決**：請將該圖片拖入白色背景的軟體中（如 Word、Canva 白底畫布），QR Code 就會立刻顯現。
* **線上掃描注意**：許多線上掃描器不支援透明圖層，會導致讀取失敗。**若要使用線上工具驗證，建議生成時暫時不要勾選透明背景，或使用白底的 JPG。**

### 使用第三方工具驗證 (qrcoderaptor.com)

我們推薦使用獨立的第三方工具來驗證內容的純淨度：

1.  在本工具中生成一張**帶有白色背景**的 QR Code (未勾選透明) 並下載。
2.  前往線上掃描器：[https://qrcoderaptor.com/](https://qrcoderaptor.com/)
3.  點擊 "Upload Image" 並上傳您剛剛下載的圖片。
4.  查看結果：
    * ✅ **正常**：掃描結果直接顯示您的目標網址（如 `https://memory8.com`）。這證明了它是直連的靜態碼。
    * ❌ **被綁架 (其他付費競品)**：如果顯示的是奇怪的短網址（如 `qrfy.com/...`），代表該 QR Code 有過期風險。
